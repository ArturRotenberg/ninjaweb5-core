# 🥷 NinjaWeb5 — Programmable Web Infrastructure

Control SEO, QA, security, and UI at the edge — without backend changes.

---

## 🚀 What is NinjaWeb5?

NinjaWeb5 is a programmable delivery-time control layer for the web.

It allows you to modify HTML, scripts, content, and behavior **between origin and browser**.

---

## ⚡ What you can do

- 🔍 Fix SEO without touching backend
- 🧪 Control production with QA rules (canary, rollback)
- 🔐 Manage scripts and protect data in the browser
- 🎯 Run experiments and UI changes without deployments
- 🧱 Inject dynamic content via datasets

---

## 💡 Example

```js
// Change title dynamically
if (ctx.url.includes('/product')) {
  ctx.html.replace('title', 'Best Product 2026')
